## Download Patch

### Ubuntu & RHEL

- Download tar package. The link will be shared by the WaveMaker team.

```bash
wget  <WME-Patch-Link>
```

## Copy Patch to Platform Machine

- Copy downloaded patch file to the home directory on the Platform Instance.
