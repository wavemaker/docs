*WaveMaker released another version update with new features and bug fixes.*

---

## Bug Fixes

- Fixed an issue related to the Dynamic table's search dropdown. Columns will show up in the search dropdown, and it works as expected.
- Fixed an issue related to the 'Is Null' filter condition of the Database CRUD Variable, not working at the runtime.
