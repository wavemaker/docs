*WaveMaker has released a new version with enhancements and bug fixes.* 

## Fixes
---

- Rating Widget: Sticky hover issues on mobile are now fixed. Due to this the rating selection on widget did not work properly.
- CSS styles of one page were affecting the styles on another page. This is now fixed.
