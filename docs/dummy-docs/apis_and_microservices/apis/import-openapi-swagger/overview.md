# Overview

Import and integrate external APIs using OpenAPI/Swagger specifications and WebSocket connections. This section covers everything you need to consume and integrate third-party services.

## Key Features

- **OpenAPI/Swagger Import**: Automatically generate client code from API specifications
- **REST Services Integration**: Connect to RESTful web services
- **WebSocket Support**: Real-time bidirectional communication
- **OAuth 2.0**: Secure authentication for API access

## What You'll Learn

- How to import and use OpenAPI/Swagger definitions
- Configuring REST service clients
- Implementing secure server-side properties
- Setting up request timeouts
- Using OAuth 2.0 for authentication
- Establishing WebSocket connections

## Benefits

- Faster integration with external services
- Type-safe API clients
- Standardized error handling
- Built-in security features
- Real-time communication capabilities

## Common Use Cases

- Integrating payment gateways
- Consuming third-party APIs
- Real-time notifications
- Microservices communication
- External service authentication
