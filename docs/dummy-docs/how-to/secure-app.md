# How to Secure Your App

Security is paramount. WaveMaker provides built-in support for Spring Security.

## Enable Security

1.  Open **Project Settings** > **Security**.
2.  Toggle **Enable Security**.

## Authentication Providers

You can configure various providers:

*   **Database**: Authenticate against your own database users table.
*   **LDAP/AD**: Integrate with corporate directories.
*   **OIDC/OAuth2**: Use providers like Google, Azure AD, or Okta.
*   **SAML**: Enterprise SSO.
*   **CAS**: Central Authentication Service.
