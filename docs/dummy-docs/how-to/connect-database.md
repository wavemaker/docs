# How to Connect to a Database

Connecting to an external database is a core feature of WaveMaker.

## Steps

1.  Navigate to the **Databases** section in the Studio.
2.  Click on **Import Database**.
3.  Choose your database provider (MySQL, PostgreSQL, Oracle, SQL Server).
4.  Enter the connection details:
    *   **Host**: The database host URL.
    *   **Port**: The port number.
    *   **Username**: Database username.
    *   **Password**: Database password.
5.  Test the connection and click **Import**.

## Supported Databases

*   MySQL
*   PostgreSQL
*   Oracle
*   SQL Server
*   DB2
