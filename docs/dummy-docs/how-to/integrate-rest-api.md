# How to Integrate a REST API

WaveMaker makes it easy to consume REST APIs.

## Import Web Service

1.  Go to **Web Services** > **Import**.
2.  Select **REST**.
3.  Provide the URL (e.g., Swagger/OpenAPI URL or a direct endpoint).
4.  Configure authentication (Basic, OAuth, etc.) if required.
5.  Review the endpoints and click **Import**.

## Binding to UI

Once imported, a **Service Variable** is created. You can bind the output of this variable to any widget, like a List or Data Table.
