# VCS Manager

The VCS Manager in WaveMaker provides a unified workspace to manage your application’s source code and its evolution over time. It brings common version control operations—such as pulling changes, pushing commits, reviewing modifications, and tracking history—directly into the WaveMaker UI, enabling a smooth workflow for teams working in parallel.

This workspace allows you to clearly see what has changed, who made those changes, and how different versions of the same file compare. It also helps you safely synchronize local work with remote repositories, handle merge conflicts when they arise, and connect your project to external version control systems like GitHub or Bitbucket.

To get familiar with the layout, options, and workflows available in this workspace, watch the **[VCS Manager Walkthrough video](https://app.guidde.com/share/playbooks/9W5WHiWsENAqc4AzErVkmS?origin=UHEVgZL3MXReLMMFtXRKKLRtXmG3)**. The video walks through the UI step by step, helping you understand how WaveMaker presents version control and supports collaborative development before you start using it in your day-to-day work.
