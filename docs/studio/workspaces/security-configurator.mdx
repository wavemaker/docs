# Security Configurator

The Security Configurator in WaveMaker is a centralized workspace designed to help you define, manage, 
and fine-tune the security of your application. It brings together all major security-related
configurations, ranging from transport-level security and safeguards to authentication, authorization, 
and role-based access control, into a single, intuitive interface.

This configurator allows you to visually explore and configure how your application handles secure
communication, protects against common web vulnerabilities, integrates with identity providers,
and controls access to pages, services, and prefabs. It presents a structured and cohesive view of
application security.

To help you become familiar with this workspace, we recommend watching the **[Security Configurator 
Walkthrough video](https://app.guidde.com/share/playbooks/fuAUQXoo4xSoZ8Nuo5VbD2?origin=UHEVgZL3MXReLMMFtXRKKLRtXmG3)**. This video provides a guided tour of the UI, explaining where each configuration 
lives and how the different sections relate to one another. It is the ideal starting point before you 
begin configuring security for your application.
