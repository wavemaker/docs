# WaveMaker Event Lifecycle 

## Overview
WaveMaker follows a layered, event-driven architecture where Pages, Variables, and UI Components communicate through events, state changes, and programmable methods. 

The lifecycle flows top-down and bottom-up, depending on whether the trigger is system-driven, API-driven, or user-driven.
