# Common Events for Components

WaveMaker components provide a rich set of events to handle user interactions, data changes, and UI state transitions.
While each component (Form, DataTable, List, Card, Button, etc.) exposes component-specific events, many components also share a set of common events that are used frequently across applications.

This section focuses on the most commonly used events, explains when they are triggered, and highlights typical use cases where these events are applied in real applications.

If you want to explore all available events for a specific component, refer to the individual [Component documentation](https://react-components.wavemaker.com).

- **Form Component Events**
    - **onBeforeSubmit:** Fires just before the form data is submitted to a variable or API.
            ```javascript
                Page.userFormBeforeSubmit = function ($event, widget) {
                    // ex-1: Basic validation
                    if (!Page.Variables.UserForm.data.email) {
                        alert("Email is required");
                        return false; // blocks form submission
                    }

                    // ex-2: Trim input data
                    Page.Variables.UserForm.data.username =
                    Page.Variables.UserForm.data.username.trim();
                };
            ``` 

    - **onSubmit:** When the submit action starts (request is triggered).
            ```javascript
                Page.userFormSubmit = function ($event, widget) {
                    // ex-1: Disable submit button
                    widget.disable = true;

                    // ex-2: Show loading indicator
                    Page.isSubmitting = true;
                };
            ``` 

    - **onSuccess:** After successful form submission and server response.
            ```javascript
                Page.userFormSuccess = function ($event, widget, response) {
                    // ex-1: Refresh DataTable
                    Page.Variables.UserList.invoke();

                    // ex-2: Reset form
                    Page.Widgets.userForm.reset();
                };
            ``` 

- DataTable Events
    - **onRowSelect:** When a user selects a row in table.
            ```javascript
               Page.userTableRowSelect = function ($event, widget, rowData) {
                    // ex-1: Store selected row
                    Page.selectedUser = rowData;

                    // ex-2: Load details
                    Page.Variables.UserDetails.setInput('id', rowData.id);
                    Page.Variables.UserDetails.invoke();
                };
            ``` 

    - **onDataSort:** When sorting is applied on a column.
            ```javascript
               Page.userTableDataSort = function ($event, widget, sortInfo) {
                    // ex-1: Pass sorting info to backend
                    Page.Variables.UserList.setInput('sortBy', sortInfo.field);
                    Page.Variables.UserList.setInput('sortOrder', sortInfo.direction);

                    // ex-2: Fetch sorted data
                    Page.Variables.UserList.invoke();
                };
            ``` 
    - **onHeaderClick:** When any column header is clicked.
            ```javascript
                Page.userTableHeaderClick = function ($event, widget, column) {
                    // ex-1: Show column help
                    console.log("Header clicked:", column.field);

                    // ex-2: Toggle filter panel
                    Page.showFilterPanel = true;
                };
            ``` 

- List Events
    - **onRender:** Fires each time the list is rendered/refreshed.
            ```javascript
                Page.userListRender = function ($event, widget) {
                    // ex-1: Highlight first item
                    if (widget.data && widget.data.length > 0) {
                        Page.selectedItem = widget.data[0];
                    }

                    // ex-2: Apply conditional styling
                    console.log("List rendered with", widget.data.length, "items");
                };
            ``` 
    - **onSelect:** When a list item is selected.
            ```javascript
                Page.userListSelect = function ($event, widget, selectedItem) {
                    // ex-1: Store selection
                    Page.selectedUser = selectedItem;

                    // ex-2: Load related data
                    Page.Variables.UserOrders.setInput('userId', selectedItem.id);
                    Page.Variables.UserOrders.invoke();
                };
            ``` 
    - **onReorder:** When list items are rearranged (drag & drop).
            ```javascript
                Page.userListReorder = function ($event, widget, reorderedList) {
                    // ex-1: Update sequence numbers
                    reorderedList.forEach(function (item, index) {
                        item.order = index + 1;
                    });

                    // ex-2: Save order to backend
                    Page.Variables.SaveUserOrder.setInput('users', reorderedList);
                    Page.Variables.SaveUserOrder.invoke();
                };
            ``` 
- Card Events
    - **onClick:** When the card is clicked.
            ```javascript
                Page.userCardClick = function ($event, widget, item) {
                    // ex-1: Navigate to details page
                    App.Actions.goToPage_UserDetails({ userId: item.id });

                    // ex-2: Set active card
                    Page.activeCardId = item.id;
                };
            ``` 
    - **onMouseOver:** When cursor hovers over card.
            ```javascript
                Page.userCardMouseOver = function ($event, widget, item) {
                    // ex-1: Show tooltip
                    Page.hoveredUserName = item.name;

                    // ex-2: Apply hover effect
                    widget.addClass("card-hover");
                };
            ```
