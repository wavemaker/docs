# Overview

## What is a Layout?

A layout defines the structural arrangement of content on a page. It acts as the foundation for building any user interface by determining where and how components are placed on the screen. To structure a page in the UI or position components in the desired locations, layouts play a crucial role. Without layouts, managing alignment, spacing, and responsiveness becomes difficult, especially as the page grows in complexity.

In WaveMaker, a wide range of layout components is available to help arrange page content efficiently. These layouts organize components into meaningful sections, ensuring consistency, responsiveness, and ease of maintenance across the application. By using layouts, you can achieve consistent UI patterns across pages and applications while also building responsive designs with minimal effort.

WaveMaker supports multiple layout options to address different design needs. 
- Container 
- Accordion 
- Panel
- Tabs
- Tile
- Wizard. 

Each layout is designed to simplify UI development and improve the overall user experience.

---

## Page Content Layout

While the **Page Layout** defines the overall page structure, **Page Content Layouts** control how components are organized within the page content area. WaveMaker provides several layout components to simplify content organization and improve user experience.
