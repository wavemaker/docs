### Overview

Camera Widget can be used to capture images and videos from a Mobile App In this post, we will create a mobile app to capture and display a photo. Image: camera_run3.png

1. Create or Open a Hybrid Mobile App.
2. Create a page choosing  and _blank template_ and ‘_One column layout with top navbar_’ Image: camera_template.png
3. Drag and drop a _container widget_ and set horizontal align property as center.
4. Drag and drop a camera widget.
5. Drag and drop a picture widget and set its width as 100%. Image: camera_design.png
6. Bind the source of picture to the ‘dataValue’ of camera widget. Image: camera_binding.png

1. Build the apk, install it on a android phone and open the app. Image: camera_run1.png
2. Open the app and click on camera button and take a photo. Image: camera_run2.png
3. The captured image will get displayed in the app. Image: camera_run3.png
