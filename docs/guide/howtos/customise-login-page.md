A Login page is added to your app, by default.

1. You can select from the list of default templates and change it for this page, know more
2. You can make modifications to the login page components:
    - Changing the app logo will change the logo for this page, too
    - You can access the Login page from Pages Resource Image: design_login-1.png
    - Select the _page container_ and change the _Class name_ and _Image_ location from the _Style_ property to change the background image and style Image: design_login_style.png
