## WaveMaker List of Agents

1. WaveMaker Agent – Workflow coordinator
2. Architect Agent – Documentation lookup + architecture guidance
3. UI Expert Agent – Declarative UI markup generation
4. Prism UI Expert Agent – Visual/UI generation with token-aware styling
5. JavaScript Agent – Safe WaveMaker JS behaviors
6. Theme Agent – Design tokens + theme JSON
7. WMX Agent – React Native WMX components
8. Java Agent – Backend Java services
9. Database Agent – DB inspection + Query Services
10. API Agent – API import/update
11. Marketplace Agent – Prefabs, connectors, WMX installs
12. API Binding Agent – Autonomous widget-to-API binding
13. Security Agent – Auth + RBAC + security config
14. i18n Agent – Localization and formats
15. VCS Agent – Git operations
16. Rescue Agent – Log analysis + remediation workflow
